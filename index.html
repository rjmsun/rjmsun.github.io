How the Algorithm Works

Expected Information Optimization: The solver calculates how much information each possible guess provides and recommends words that maximize learning.

Information Scoring:

  • Green (1.5 pts) - Exact position known, highest value
  • Yellow (1.0 pts) - Letter in word, wrong position
  • Grey (0.4 pts) - Letter not in word, still informative

What the Numbers Mean:

  • Words Remaining: Average number of possible answers left after this guess
  • New Letters: How many untested letters this word explores
  • ANSWER Badge: This word could be the actual solution
  • Unique Letters: Ratio of unique letters (5/5 is best for exploration)

Strategy: Early guesses focus on information gathering using the full 25,500+ word database. Later guesses target the remaining possible answers from the 14,855 valid solutions.
